## Project Overview

Real time chap app with following features:

1. Personal Profile
1. Contacts
1. Private Chat with Contacts
1. Public Chat Room
1. Ability to add / remove contacts
1. Chat history

## Stack

* Angular 4
* Nodejs and Express
* Mongodb + Socket.io
* Bootstrap 4.0

## Dependencies

1. Angular 4.0 and its dependencies
1. Bootstrap 4.0

1. Mongodb
1. Socket.io
